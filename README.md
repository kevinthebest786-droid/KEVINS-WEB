# KEVINS-WEB




Importance of Fasting in Islam
A clean, responsive educational website built using HTML and Tailwind CSS to explain the spiritual, social, and moral significance of fasting (Sawm) in Islam.


Overview
This project is a static educational website created to highlight the importance of fasting in Islam, particularly during the month of Ramadan.
The website focuses on presenting meaningful religious content in a structured, modern, and user-friendly layout while demonstrating front-end development skills.
It combines clean design principles with responsive web development using Tailwind CSS.


Features
Responsive design for mobile, tablet, and desktop
Modern navigation bar
Hero section with gradient background
Structured content sections
Card-based layout for explanations
Quranic reference section
FAQ section
Smooth hover effects and transitions
Clean typography and spacing
Fully static (no backend required)



Design Approach
The website uses a minimal and readable layout to ensure users can focus on the educational content. Tailwind CSS utility classes are used to maintain consistent spacing, typography, and responsiveness.
The design prioritizes clarity, accessibility, and structured information hierarchy.


Technologies Used
HTML5
Tailwind CSS (CDN version)


How to Use
Download or clone the repository.
Open the HTML file in any modern browser (Chrome, Safari, Edge, Firefox).
No installation or dependencies required.
